Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: In JavaScript innerHTML, innerText and textContent all of these used for manipulating html elements. So from there key differences written below:
innerHTML: Includes all HTML tags within the element, Useful for manipulating the full HTML structure of an element, Can be used to inject new HTML content into an element.
innerText: Only returns the visible text content of an element, Ignores hidden elements and CSS styling, Best used when we only need the text that a user would see on the page.
textContent: Retrieves all text content within an element, including hidden text. Preserves whitespace and line breaks. Useful for extracting all text content from a page, even if it's not visually displayed.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.

Question-4: What is event bubbling in the DOM?

Answer: Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: In JavaScript, we can create, add, and remove elements using the DOM.

To create an element.
Use document.createElement(tagName)


Adding an Element Once created, we can add the element to the DOM using:
appendChild() → Adds as the last child
append() → Can add text nodes along with elements
prepend() → Adds as the first child
insertBefore() → Adds before a reference node
replaceChild() → Replaces an existing child


We can remove an element using:
removeChild() → Removes a specific child from the parent
remove() → Directly removes an element
